# 정보처리기사 실기 - C언어 중첩 반복문과 switch 조건 문제 풀이

## ✅ 문제 19

### 📘 문제 설명  
다음은 C언어 코드이다. 출력 결과를 쓰시오.

```c
#include <stdio.h>
int main(){
    int i, j;
    for(i = 1; i < 3; i++){
        for(j = 1; j <= i; j++){
            switch((i + j) % 2){
                case 0:
                    printf("%d", i + j);
                    break;
                case 1:
                    printf("%d", i - j);
            }
        }
    }
    return 0;
}
```

---

### 🔎 문제 분석

중첩 반복과 `switch`를 통해 `(i + j) % 2`에 따라 출력이 달라짐

| i | j | i+j | (i+j)%2 | 출력 내용 |
|---|---|-----|----------|-----------|
| 1 | 1 | 2   | 0        | 1+1 = 2   |
| 2 | 1 | 3   | 1        | 2-1 = 1   |
| 2 | 2 | 4   | 0        | 2+2 = 4   |

---

### ✅ 최종 출력 결과

```
214
```

> 반복 구조에 따라 `2`, `1`, `4`가 차례로 출력됨
