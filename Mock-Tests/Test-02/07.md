# 정보처리기사 실기 - Python 조건문 및 반복문 문제 풀이

## ✅ 문제 07

### 📘 문제 설명  
다음은 파이썬(Python) 코드이다. 출력 결과를 쓰시오.

```python
i = 0
sum = 0
while i < 7:
    i = i + 1
    if i % 3 == 0:
        sum -= i
    elif i % 3 == 1:
        sum += i
    else:
        sum *= i
print(sum)
```

---

### 🔎 문제 분석

- `i`는 1부터 7까지 1씩 증가
- 각 `i`에 대해 `% 3` 결과에 따라 다른 연산이 적용됨

| i | i % 3 | 연산         | sum 변화     |
|---|-------|--------------|--------------|
| 1 | 1     | sum += i     | 0 + 1 = 1    |
| 2 | 2     | sum *= i     | 1 * 2 = 2    |
| 3 | 0     | sum -= i     | 2 - 3 = -1   |
| 4 | 1     | sum += i     | -1 + 4 = 3   |
| 5 | 2     | sum *= i     | 3 * 5 = 15   |
| 6 | 0     | sum -= i     | 15 - 6 = 9   |
| 7 | 1     | sum += i     | 9 + 7 = 16   |

---

### ✅ 최종 출력 결과

```
16
```

> 조건문 순서와 연산 결과를 정확히 따라가면 최종 출력값은 `16`이 됩니다.
