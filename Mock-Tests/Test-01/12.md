### ✅ 문제 12

다음은 C언어 코드이다. 출력 결과를 쓰시오.

```
#include <stdio.h>

int main() {
    int a = 30, b = 15;
    printf("%d", a & b);
    printf("%d", a | b);
    return 0;
}
```

### 🔍 문제 분석
비트 연산자 사용 문제이다.

```
& (AND): 각 비트가 모두 1일 때만 1

| (OR): 하나라도 1이면 1
```

### 정수 30과 15의 2진수 표현

```
a = 30 → 0001 1110  
b = 15 → 0000 1111
```

### 🧮 비트 연산 수행
a & b

```
  0001 1110 (30)
& 0000 1111 (15)
------------
  0000 1110 → 14
```

a | b
```
  0001 1110 (30)
| 0000 1111 (15)
------------
  0001 1111 → 31
```

### 🖨️ 출력 결과
```
printf("%d", a & b);  // 출력: 14
printf("%d", a | b);  // 출력: 31
최종 출력 결과: 1431
```

### ✅ 정답
```
1431
```

### 🧠 추가 개념: 비트 연산자 요약

| 연산자  | 설명              | 예시             | 
| ---- | --------------- | -------------- |
| `&`  | 비트 AND          | `30 & 15 → 14` |
| \`   | \`              | 비트 OR          |
| `^`  | 비트 XOR (배타적 OR) | `30 ^ 15 → 17` | 
| `~`  | 비트 NOT (보수)     | `~30 → -31`    | 
| `<<` | 왼쪽 시프트          | `15 << 1 → 30` | 
| `>>` | 오른쪽 시프트         | `30 >> 1 → 15` |
